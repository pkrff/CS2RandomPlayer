# CS2 Random Player

Программа для случайного распределения игроков по командам на основе скриншота лобби CS2.

## Возможности

- Загрузка скриншота (PNG, JPG) лобби CS2.
- Автоматическое распознавание никнеймов с помощью Tesseract OCR.
- Ручное редактирование списка игроков (добавление, удаление).
- Выбор количества команд (2–10).
- Случайное перемешивание и равномерное распределение по командам.
- Простой графический интерфейс на Tkinter.
- Возможность указать путь к Tesseract вручную.

## Системные требования

- Windows 7/10/11
- Установленный [Tesseract OCR](https://github.com/UB-Mannheim/tesseract/wiki) (или портативная версия рядом с EXE)

## Установка

1. Скачайте последнюю версию `CS2RandomPlayer.exe` из раздела [Releases](https://github.com/yourusername/CS2RandomPlayer/releases).
2. Установите Tesseract OCR (ссылка выше) **или** поместите папку `tesseract` с `tesseract.exe` в одну директорию с EXE.
3. Запустите `CS2RandomPlayer.exe`.

## Использование

1. Нажмите **«Загрузить скриншот»** и выберите изображение лобби CS2.
2. Проверьте распознанные никнеймы в списке. При необходимости отредактируйте:
   - Удалите лишние строки (выделите и нажмите `Delete`).
   - Добавьте игрока кнопкой **«Добавить игрока вручную»**.
3. Нажмите **«Подтвердить список»**.
4. Выберите количество команд (по умолчанию 2).
5. Нажмите **«Перемешать и распределить»**.
6. Результат отобразится в текстовом поле.

## Настройка Tesseract

Если программа не может найти `tesseract.exe`, нажмите **«Настройки»** и укажите путь вручную.
