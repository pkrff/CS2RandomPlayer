# CS2 Random Player

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![Release](https://img.shields.io/badge/release-v1.1.0-blue)](https://github.com/pkrff/CS2RandomPlayer/releases)

Программа для случайного распределения игроков CS2 по командам на основе скриншота лобби.  
**Больше не нужно вводить ники вручную** — используйте встроенный помощник распознавания текста через Яндекс.Картинки.

---

## 📌 Особенности

- 🖼️ **Удобное распознавание**: загрузите скриншот в Яндекс.Картинки, скопируйте текст и вставьте в программу.
- ✏️ **Ручное редактирование**: проверьте список, добавьте или удалите игроков.
- 🎲 **Случайное распределение**: выберите количество команд (от 2 до 10) и получите честный рандом.
- 💻 **Портативный EXE**: не требует установки Python или дополнительных библиотек.
- 🧩 **Простой интерфейс**: минимум действий, максимум удобства.

## 🚀 Как использовать

1. **Запустите программу** `CS2RandomPlayer_v1.1.0.exe`.
2. Нажмите кнопку **«🌐 Открыть Яндекс.Картинки (OCR)»**.
3. В открывшемся браузере:
   - Перетащите скриншот лобби CS2 в окно Яндекс.Картинок.
   - Нажмите на иконку **«Распознать текст»** (или дождитесь автоматического выделения).
   - Скопируйте весь распознанный текст (`Ctrl + C`).
4. Вернитесь в программу и **вставьте текст** в большое поле (`Ctrl + V`).
5. Нажмите **«📋 Разобрать список»** — никнеймы появятся в списке.
6. Проверьте список, при необходимости:
   - Добавьте игрока вручную ➕
   - Удалите лишнего ❌
   - Подтвердите список ✅
7. Выберите количество команд (по умолчанию 2) и нажмите **«🎲 Перемешать и распределить»**.
8. Результат отобразится в нижнем поле — готово!
